# daysvane-Mandela-TechQuest
Mandela TechQuest es una plataforma educativa interactiva diseñada para fortalecer el aprendizaje de Tecnología e Informática mediante misiones, retos, juegos, videos y actividades gamificadas para estudiantes de 6° a 11° de la Institución Educativa Mandela. Aprende. Supera retos. Construye el futuro.
